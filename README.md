2.2	基于Tableau的某超市销售数据报告
2.2.1	第一部分
2.2.1.1	各省销售额和类别堆积条形图
 
堆积柱形图用来显示单个项目与整体间的关系，比较各个类别的每个数值占总数值的大小；
相比于饼图，他能够呈现更多维度和类型的数据的部分和整体之间的关系。
相比于散点图，他虽然也能呈现很多维度的数据，但是散点图不够直观，难以理解数据之间的比例。
综合考虑，堆积条形图是一个很好的选择
 

2.2.1.2	各类别销售量直方图
 
相比于之前的柱状图，直方图的优势就在于能够清晰地展示数据的分布情况。
能够很以展示数据的分布情况，诸如众数、中位数的大致位置、数据是否存在缺口或者异常值。
比如这幅图，就能清楚地看到，众数就是0k这个段位，中位数大概在4k左右。
而且数据存在缺失值，分别在18k，19k，22-24k都存在缺失。
2.2.1.3	不同类别商品销售额饼图
 
饼图能够通过扇形面积和弧度展示数据的比例关系，具有很好的直观性。
但是同时饼图也具有很大的局限性，比如他只能展示一个维度的数据，要展示多个维度的数据，虽然有堆积饼图，但是效果往往不是很理想。
而且他不能展示一个维度很多类别的数据，当类别增加的时候，很多数据值较小的数据在饼图中变得越来越小，最后无法呈现出来。
2.2.1.4	月销售额趋势折线图

 
折线图可以清晰地显示出数据在一个连续的时间间隔或者时间跨度上的变化趋势。通过观察折线图的走向，我们可以了解事物随时间或有序类别的变化情况。
在此图中，我们通过折线图，可以很明显地看到，七月份的销售额有了一个巨大的下降，随后在8月销售额又达到了峰值。同时10-12月，销售额开始趋于平缓，于是我们可以针对这两个问题来进行具体的分析。
 

2.2.1.5	各省利润突显表
 
凸显表通过不同的颜色、字体、加粗等方式，将表格中的信息进行层次划分，使读者能够更好地理解数据的意义和关联。
比如在本图中，将总利润为负数的值，用橙黄色高亮处理；将总利润较高的值，用深蓝色高亮处理。视觉效果直观，可读性强。
 


2.2.1.6	各省销售额和利润树形图
 
树形图相比于凸显表，它增加了一个大小的维度。所以可以将维度字段用颜色来处理，将度量字段用大小来处理。可以展示数据量较大的，多维度的数据，同时还能使得层次分明，可读性强。
通过本图，我们可以发现，存在销售额和利润都比较大的数据，比如：山东。
也存在销售额和利润都比较小的数据，比如：甘肃。
但是我们还能发现一些特殊的数据：比如辽宁，他在销售额较大的同时，利润比较小。于是我们便可以针对他来进行数据分析，来发现问题。
 

2.2.1.7	各省销售额气泡图
 
气泡图可以展示三个数值型变量之间的关系。在绘制的时候，把变量X放在横轴，变量Y放在纵轴，变量Z用气泡的大小来表示。
同时，气泡图比散点图更为复杂，因为它们还包括一个附加的维度——气泡大小，用于代表第三个数值数据。所以各省销售额的数据，使用气泡图来表示更为恰当。
 

2.2.1.8	各子类别在各省市的销售额圆视图
 
圆视图通过给气泡图添加一个相关的维度，按不同的类别分析气泡，并依据度量的大小，将所有气泡有序地排列起来，表现较气泡图更为清晰。
圆视图相比于气泡图，它对于离散型的数据具有更佳的表现情况，可以帮助分析人员快速发现每一类别中的异常点或突出数据。
比如：书架在山东的销售额，就异常地高。
2.2.2	第二部分
2.2.2.1	各省销售计划完成情况标靶图
 
标靶图是指通过在基本条形图上添加参考线和参考区间，可以帮助分析人员更加直观地了解两个度量之间的关系，常用于比较计划值和实际值。
本图中，我们可以通过标靶图中所添加的表示计划的红线，清晰地看出各省对于计划的完成情况。
2.2.2.2	2016年各省商品交货情况的甘特图
 
甘特图以图示的方式通过活动列表和时间刻度形象地表示出任何特定项目的活动顺序和持续时间。
甘特图可以用来比较与日期相关的不同活动的持续时间长短，也常用于显示不同任务之间的依赖关系。
本图中，单维度来看，可以看出安徽省全年的延迟交货天数都较少。
多维度来看，可以看出各省在一月普遍延迟交货较少，然而在11月，12月延迟交货天数普遍增加。
 

2.2.2.3	各地区平均销售额统计的盒须图、各地区酒店不同价格统计的盒须图

 
盒须图用以显示数据的位置、分散程度、异常值等。箱线图主要包括6个统计量：下限、第一四分位数、中位数、第三四分位数、上限和异常值。
本图可以看出，沙田，离岛等地区，对于数据的聚合程度较好，然后油尖旺地区，存在很多异常值，很多数据点的价格十分高。
2.2.2.4	不同子类别盈亏的瀑布图
 
瀑布图，主要用来表现数据的增减变化或数据间的差异。
但是他具有很大的局限性，适用场景相对单一：瀑布图主要适用于解释两个数据值之间的差异。对数据的要求相对较高
2.2.2.5	流量转化漏斗图
 
漏斗图，适用于业务流程的比较，是业务流程的直观展现形式，可以发现流程中的异常。
本图中，漏斗展示了方面，相对于上一阶段的百分比和相对于第一阶段的百分比。
我们可以看出，在公众号访问--关注阶段，和关注-客服咨询阶段，流失人数比例较多。但是在客服咨询-成交单数阶段，流失人数较少，可以发现，客服咨询这个阶段做的比较好。
2.2.2.6	不同子类别利润凹凸图
 
通常用在不同排名中连接相同事物以显示排名间相互变化关系。
从图中可以看到，对于桌子利润，一直是排名17位
但是也有变化比较大的，比如附件：从第7位到第3位，再到第6位和第五位。
2.2.2.7	各省市销售量和利润的符号地图
 
符号地图即以地图为背景，在对应的地理位置上以多种形状展示信息，为使用符号地图功能。
符号地图可以展示数据再真实空间状态下的分布情况和数据情况，使得数据展示更加直观，易于理解。
2.2.2.8	喜剧、剧情、爱情、动作电影动态气泡图
 
可以同时显示多个数据维度。通常，这种图表会在两个数值轴上显示数据，一个用于表示气泡的大小，另一个用于表示气泡的位置。
能够很好地反映数据之间的相对大小关系，同时结合动画，可以为其加上一个时间维度，用于表现数据随着时间的变化情况。

2.3.4	多维地图
 
多维度地图通过对不同维度的信息用多个地图展示，实现信息的分维度比对。多维度地图展示要在已创建好的符号地图或地图的基础上创建。
本图展示了在时间维度和类别维度下各省/自治区的销售额的大小。

2.3.5	混合地图
混合地图是指把符号地图和填充地图叠加的一直地图形式。
本图既有符号地图的特点-使用点的大小来表示利润的大小；也有填充地图的特点-用颜色的深浅来表示销售额的多少。
